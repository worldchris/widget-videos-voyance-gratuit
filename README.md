# Widget Vidéos : Intuition & Ancrage (Christophe Voyance)

Bienvenue sur le dépôt officiel du widget vidéo "Intuition Libre". 

Ce projet open source permet aux webmestres d'intégrer gratuitement et facilement un moteur de recherche de vidéos dédié au développement personnel, à l'éveil de l'intuition et à l'ancrage énergétique sur leur propre site web.

## 🌟 À propos de ce widget

Dans un esprit de bienveillance et de divertissement, ce widget propose des contenus vidéo structurés pour aider chacun à se recentrer face aux défis du quotidien. 

**Important :** Notre approche est pragmatique et vise l'autonomie de réflexion. Il n'y a ici aucun dogme sectaire. Les contenus diffusés via ce widget n'ont aucune vocation médicale ou thérapeutique et ne remplacent en aucun cas l'avis d'un professionnel de la santé.

## 🚀 Fonctionnalités clés

* **100% Gratuit :** Aucune inscription ni clé API n'est nécessaire.
* **Respect de la vie privée (RGPD) :** Le lecteur vidéo utilise le domaine `youtube-nocookie.com` pour garantir la confidentialité de vos visiteurs.
* **Responsive Design :** S'adapte parfaitement aux sidebars, aux articles de blog et aux écrans mobiles.
* **Léger :** Conçu pour ne pas impacter le temps de chargement de vos pages.

## 💻 Comment installer ce widget sur votre site ?

Pour éviter toute erreur de formatage ou de code corrompu, nous avons créé une page dédiée qui génère le code d'intégration HTML exact, prêt à être copié-collé.

👉 **Obtenez le code d'intégration ici :** [https://worldchris.github.io/widget-videos-voyance-gratuit/](https://worldchris.github.io/widget-videos-voyance-gratuit/)

Il vous suffit de copier l'extrait fourni sur cette page et de le placer dans le code HTML de votre site (dans un widget texte WordPress, une page HTML classique, etc.).
